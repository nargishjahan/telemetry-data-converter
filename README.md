# Telemetry Data Converter
This project converts telemetry data from multiple input formats into a unified JSON structure. It handles differences in schema, field naming, and timestamp formats (ISO to milliseconds).
## Features
- Supports multiple telemetry input formats
- Converts ISO timestamps to epoch milliseconds
- Normalizes nested and flat JSON structures
- Includes unit tests for validation
## Tech Stack
- Python
- JSON processing
- unittest
